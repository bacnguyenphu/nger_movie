# Nger Movie Platform 🎬

A movie streaming website built to discover films, search for specific titles, and view detailed movie information using external APIs.

**Timeline:** 09/2024 - 10/2024

## 🔗 Links
* **Live Demo:** [Nger Movie on Vercel](https://nger-movie.vercel.app/)
* **GitHub Repository:** [nger_movie](https://github.com/bacnguyenphu/nger_movie)

## 🚀 Technologies Used
* **Core:** ReactJS
* **Styling:** Tailwind CSS

## ✨ Key Features
* **Movie Discovery:** Display comprehensive and updated lists of movies.
* **Search Functionality:** Quickly search for specific movies by title.
* **Pagination:** Seamlessly navigate through multiple pages of movie results.
* **Detailed Views:** Access dedicated pages containing in-depth information for each movie.
